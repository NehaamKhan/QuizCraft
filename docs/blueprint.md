# **App Name**: QuizCraft AI

## Core Features:

- Concept Extraction: Automatically extract key concepts from input educational texts using AI.
- Hierarchical Organization: Organize the extracted concepts into a hierarchical structure.
- Quiz Question Generation: Generate 10 quiz questions based on the extracted concepts using a tool for intelligent question selection. LLM reasons whether to extract concepts and context from the available resources.
- Difficulty Ranking: Rank the generated quiz questions by difficulty level.
- Difficulty Validation: Validate the difficulty ranking logic using a self-checking mechanism.
- User Interface: Present a clean, intuitive user interface for text input and quiz display.
- Quiz Display: Display the quiz questions and answer options to the user, ranked by difficulty.

## Style Guidelines:

- Primary color: HSL(210, 70%, 50%) which converts to #3399FF. This vibrant blue evokes a sense of trust, intelligence, and progress.
- Background color: HSL(210, 20%, 95%) which converts to #F0F8FF. A very light blue, almost white, providing a clean and non-distracting backdrop.
- Accent color: HSL(180, 60%, 40%) which converts to #33CCCC. A contrasting teal that complements the blue, drawing attention to important elements and calls to action.
- Body and headline font: 'Inter', a sans-serif font known for its clean, readable design.
- Use clear, minimalist icons to represent different question types and difficulty levels.
- Maintain a clean and organized layout with clear visual hierarchy to guide users through the text input and quiz-taking process.
- Incorporate subtle animations to provide feedback and enhance the user experience.
